15"""Records previous preprocessing operations and allows them to be repeated.
16
17Used with object_detection.core.preprocessor. Passing a PreprocessorCache
18into individual data augmentation functions or the general preprocess() function
19will store all randomly generated variables in the PreprocessorCache. When
20a preprocessor function is called multiple times with the same
21PreprocessorCache object, that function will perform the same augmentation
22on all calls.
23"""

74 def get(self, function_id, key):
75 """Gets stored value given a function id and key.
76
77 Args:
78 function_id: identifier for the preprocessing function used.
79 key: identifier for the variable stored.
80 Returns:
81 value: the corresponding value, expected to be a tensor or
82 nested structure of tensors.
83 Raises:

85 """
