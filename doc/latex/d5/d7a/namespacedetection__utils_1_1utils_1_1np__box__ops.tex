{\footnotesize\ttfamily detection\+\_\+utils.\+utils.\+np\+\_\+box\+\_\+ops.\+ioa (\begin{DoxyParamCaption}\item[{}]{boxes1,  }\item[{}]{boxes2 }\end{DoxyParamCaption})}

\begin{DoxyVerb}Computes pairwise intersection-over-area between box collections.

Intersection-over-area (ioa) between two boxes box1 and box2 is defined as
their intersection area over box2's area. Note that ioa is not symmetric,
that is, IOA(box1, box2) != IOA(box2, box1).

Args:
  boxes1: a numpy array with shape [N, 4] holding N boxes.
  boxes2: a numpy array with shape [M, 4] holding N boxes.

Returns:
  a numpy array with shape [N, M] representing pairwise ioa scores.
\end{DoxyVerb}
